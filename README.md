# 欢迎页面项目

这是一个简单的欢迎页面项目，包含基础的HTML、CSS、JavaScript和PHP元素。

## 功能特点

- 响应式设计，适应不同屏幕尺寸
- 深色/浅色主题切换功能
- 使用PHP显示动态内容（如当前时间）
- 简单的动画和过渡效果

## 目录结构

```
├── index.php         # 主页面
├── css/              # 样式文件
│   └── style.css     # 主样式表
├── js/               # JavaScript文件
│   └── script.js     # 主脚本文件
└── README.md         # 项目说明
```

## 如何使用

1. 确保您的服务器支持PHP
2. 将所有文件上传到服务器
3. 通过浏览器访问index.php

## 自定义

您可以通过修改以下文件来自定义页面：

- `css/style.css` - 更改外观和样式
- `js/script.js` - 添加或修改交互功能
- `index.php` - 更改页面结构和内容

## 许可

此项目仅供学习和个人使用。
